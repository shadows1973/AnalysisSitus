Quaoar Studio LLC // digest
We are passionate about the development of CAD/CAM/CAE applications and algorithms. Our mission is to serve manufacturers, design, and simulation engineers to create better products through smart automation. We develop software for engineers and together with engineers. We have the following technologies in our stack:
  • OpenCascade: open-source geometric modeling kernel.
  • Mobius: our in-house NURBS kernel.
  • VTK: scientific visualization.
  • Intel TBB and/or OpenMP: CPU threading.
  • Qt/Qml: user interfaces and application infrastructure.
  • SQL databases (MySQL, SQLite, etc.): data management.
  • Tcl, Python: scripting.
We develop using the following tools:
  • Visual Studio for Windows.
  • CLion on Linux.
  • Intel Parallel Studio for profiling on Windows.
  • Jira, Redmine, Mantis following our customers' preferences.
  • Git as a source control system of choice.
